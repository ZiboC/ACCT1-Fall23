Clear the canvas with background(220) before each frame to update the ball positions. Use nested for loops to check for collisions and merging of balls. The outer loop iterates through all the balls, while the inner loop iterates through balls that come before the current ball. If two balls intersect , they merge into a new ball. Check if a ball is outside the canvas and remove it from the balls array if it is.

When the mouse is clicked. It creates a new ball with random properties (position, size, speed, and color) at the mouse's current location and adds it to the balls array.
